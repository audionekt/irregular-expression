# Aurigami Design Tokens

A comprehensive catalog of all design tokens in the Aurigami design system. These tokens form the foundation of our UI components with a sophisticated color palette featuring warm neutrals and refined gold accents.

---

## Colors

The Aurigami color system is built on a two-tier architecture: **Primitive colors** provide the raw palette, while **Semantic colors** map these primitives to specific use cases.

## Usage

Import tokens in your components:

```typescript
import { colors, gradients, spacing, shadows } from '@repo/ui/tokens';

// Use in vanilla-extract
import { style } from '@vanilla-extract/css';

export const button = style({
  backgroundColor: colors.semantic.brand.primary,
  padding: spacing[4],
  boxShadow: shadows.md,
  background: gradients.brand.accent,
});
```

For themed values, use the theme contract:

```typescript
import { vars } from '@repo/ui/styles/contract.css';

export const card = style({
  backgroundColor: vars.color.semantic.background.base,
  borderColor: vars.color.semantic.border.default,
});
```
